# 四目並べ
コマンドライン上で遊べる[重力付き四目並べ](https://ja.wikipedia.org/wiki/%E5%9B%9B%E7%9B%AE%E4%B8%A6%E3%81%B9)をC++で作りました。

# 実行方法
### ビルド
```
$ git clone https://github.com/risudo/Four_in_a_row.git
$ cd Four_in_a_row
$ make
```

### 実行
```
$ ./four_in_a_row
```
